# odin-recipes

1. Git
    - Git setups: SSH keys
    - `git clone`
    - `git add`
    - `git commit`
    - `git push`
2. HTML syntax

    - `ol`, `ul`, `li`
    - `h1`, `h2`
    - `a`, `img`
3. Linux command
    - `mkdir`
    - `cd`
    - `touch`
    - `..`
